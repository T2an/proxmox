Welcome To Proxmox
==================

**Proxmox** is an open-source server virtualization management solution. It combines two virtualization technologies: KVM (Kernel-based Virtual Machine) for virtualization and LXC (Linux Containers) for lightweight container-based virtualization. Proxmox allows you to easily manage virtual machines, containers, highly available clusters, storage, and network resources through a web-based interface.

In this documentation, you will find detailed instructions on how to configure and use various services provided by Proxmox. Whether you are a beginner looking to set up your first virtual environment or an advanced user seeking to optimize your infrastructure, this guide will provide the necessary information and best practices.
